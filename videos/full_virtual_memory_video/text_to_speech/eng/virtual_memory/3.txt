For this representation of virtual addresses, <bookmark mark='POPUP_FIRST'/>the first 3 represents the offset into the top table directory. The top level directory itself contains 8 entries, which is the number of different digits you can create with 3 bits. <bookmark mark='POPDOWN_FIRST'/>. <bookmark mark='POPUP_SECOND'/> the next three will represent the offset into the second level page table, this page table will also contains 8 entries.<bookmark mark='POPDOWN_SECOND'/>. <bookmark mark='POPUP_LAST'/>The final 4 bits will represent the offset into the page <bookmark mark='POPDOWN_LAST'/> itself.
